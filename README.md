# 순수 자바 프로젝트 1

### 프로젝트 설명
- 편의점 결제 서비스(일명 ‘편결이’)
    - 사용자가 편의점에서 간편 결제로 결제할 수 있는 시스템 개발
    - 사용자 → 편결이 → 결제 수단 시스템(이미 만들어져 있다고 가정) 호출
